# gigaenterprise_site

Проект сайта gigaenterprise.ai на основе gigaenterprise-nav-v5.html.

## Структура

```
gigaenterprise_site/
├── index.html              — главная страница
├── css/
│   ├── main.css            — основные стили (nav, footer, sections)
│   └── product.css         — стили продуктовых страниц
├── js/
│   └── main.js             — форма, анимации
├── assets/
│   ├── images/             — изображения
│   └── files/              — юридические документы (PDF)
└── pages/
    ├── gigacowork.html     — страница продукта GigaCowork
    └── 404.html            — страница ошибки

```

## Запуск

Откройте `index.html` в браузере. Для корректной работы шрифтов нужен интернет
(подключается Google Fonts).

## Стек

- Чистый HTML/CSS/JS, без фреймворков
- Google Fonts: Inter + JetBrains Mono
- CSS Custom Properties для дизайн-токенов
