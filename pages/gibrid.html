Безопасность данных

Базовый — mTLS + API Gateway

  • Авторизация по HTTPS (TLS 1.3+) через API-шлюз (OAuth2/OIDC)
  • Доступ к cloud LLM по HTTPS с взаимной аутентификацией (mTLS)
  • Проверка клиентских сертификатов на уровне API-шлюза
  • PKI и ротация сертификатов

Дополнительный — Site-to-Site VPN (IPsec)

  • Приватный доступ через Site-to-Site IPsec VPN (1–3 Gbps)
  • Шифрование всего трафика между on-prem и облаком
  • Технологии: IPsec, IKEv2, BGP / статическая маршрутизация
  • Публичный интернет видит только зашифрованный IPsec-трафик
